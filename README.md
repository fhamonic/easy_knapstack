# Easy Knapstack

Optimized implementation of very simple branch and bound algorithms for 0-1 and unbounded Knapstack problems.
It is not meant to be a hard use solver but a simple and efficient one for small to medium instances.

## Dependencies
Range-v3 (https://ericniebler.github.io/range-v3/)

### Build process
The build process requires CMake 3.12 (https://cmake.org/) or more and the Conan C++ package manager (https://conan.io/).

## How to Compile
Just

    make
